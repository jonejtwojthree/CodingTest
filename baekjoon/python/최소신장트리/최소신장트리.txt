최소 신장 트리(Mininum Spanning Tree)

1. 신장 트리란
- 그래프 내의 모든 정점을 포함하는 트리
- 그래프의 최소 연결 부분 그래프
- n개의 정점을 가지는 그래프의 최소 간선수는 n-1개이고, n-1개의 간선으로 연결되어 있으면 spanning tree이다
- n개의 정점을 n-1개의 간선이 연결되어 있고 사이클이 존재하지 않음

=> 그래프의 모든 정점이 간선으로 연결되어 있는데, 간선의 수가 n-1개이면 신장 트리이다.
=> 그래프의 부분집합 중에 신장트리가 존재 
so, 1개의 그래프에 여러 개의 신장 트리가 존재한다

최소 신장 트리{kruskal알고리즘, prim알고리즘}
- 여러개의 신장 트리 중에서 가중치의 합이 최소인 신장 트리
 
1) 크루스칼 알고리즘
- greedy 방법을 이용하여 그래프의 모든 정점을 최소비용으로 연결하는 알고리즘
- 사이클 판정을 위해 (최상위)부모를 가리키는 배열이 필요함
- 부모를 찾거나 합칠때, union-find를 사용함

cf. greedy방법
- 그 순간 최고의 선택을 하는 것
- 물론 순간순간의 최고의 선택이 전체적으로 볼때 최고의 선택이지 않을 수 있음
but, 크루스칼은 greedy방법이지만 최적해가 항상 보장됨

1. 그래프의 간선들을 가중치의 오름차순으로 정렬
2. 정렬된 간선 리스트에서 순서대로 사이클을 형성하지 않는 간선을 선택
(낮은간선, + 그 간선을 추가했을때 사이클 여부 확인)

 2) 프림 알고리즘
 - 시작정점을 정하고 시작정점에서신장트리 집합을 확장해나가는 방법

 1. 시작정점을 정함
 2. MST집합에 시작정점을 포함 하고, 인접한 정점들 중에서 최소 간선으로 연결된 정점을 선택
 3. MST집합이 n-1이 될때까지 반복
